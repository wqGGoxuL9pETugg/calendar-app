Firebase ログイン

Firebase Authentication の Email/Password でログインしてから、クラウド同期された予定を開きます。 初回は同じ画面からアカウント作成もできます。

Firebase 未設定のときはこの画面は出ません。Email/Password を Firebase Console 側で有効化してから使ってください。

データを復号

このカレンダーの予定データは暗号化されています。設定したパスワードを入力すると、この端末上で復号して閲覧・編集できます。 初回は、最初に入力したパスワードで以後の保存データが暗号化されます。

Firebase 設定

設定手順：
1. Firebase Console でプロジェクトを作成
2. 「Authentication」で Email/Password を有効化
3. 「Realtime Database」を作成（テストモードで開始）
4. プロジェクト設定 → ウェブアプリを追加 → 設定値をコピー
5. 下のフィールドに貼り付けて「保存して接続」
